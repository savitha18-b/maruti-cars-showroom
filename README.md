# 🚗 Maruti Cars Showroom — Console Application

A Python console application that simulates a car dealership experience — greeting the customer, recommending cars based on budget, showing detailed specs, and finalizing the purchase.

## 💡 Features

- Welcomes the customer and offers a refreshment (tea/coffee/milk/juice)
- Takes the customer's budget and recommends matching Maruti car models
- Displays detailed price and key features for the selected car
- Finalizes the purchase with warranty, EMI, and free accessories info
- Built using **dictionaries** for clean, scalable data storage (no repetitive if-elif chains)
- Organized using **functions** for readability and reusability
- Includes **input validation** to handle invalid entries gracefully

## 🛠️ Tech Used

- Python 3
- Core concepts: dictionaries, functions, conditional logic, input handling, validation

## ▶️ How to Run

1. Make sure Python 3 is installed on your system
2. Download or clone this repository
3. Run the script:
   ```bash
   python maruti_showroom.py
   ```
4. Follow the on-screen prompts to explore the showroom
